Use /api/ehr/{ehrId} or /api/pid/{pid} to fetch patient JSON object

To update patient data, use a PUT with /api/update/{ehrId} with the data formatted:
{ "pulse": 80, "oxygen_saturation": 40, "blood_pressure_systolic": 127, "blood_pressure_diastolic": 67, "breathing_frequency": 17, "alertness": "awake", "body_temperature": 37.3 }

To authenticate a login attempt, use a GET with /api/login with the data formatted:
{ "username": "useruser", "password": "passpass" }

Use /patient_list or /staff_list to list all patients or staff members in the database
